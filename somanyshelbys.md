##### **HTML Boilerplate, Importance.**



A ready-made template for ur webpages.



Includes basic structure \& essential eleemnts eery HTML doc needs.



Saves you time and helps ensure ur pages are set up properly.



Ex:



<!DOCTYPE html>  (which tells browsers which version of HTML ur using.

<html lang="en"> (html tag wraps around all ur content, spec. lang of ur page.)

&#x20; <head>  (two main sec of html are head and a body)(head=BTS-info)

&#x20;   <meta charset="utf-8" /> (meta contains metadata, info like char. encoding, + how websites should preview ur page's link)

&#x20;   <meta

&#x20;      name="viewport"

&#x20;      content="width=device-width, initial-scale=1.0" />

&#x20;   <title>freeCodeCamp</title> (title determines text in browser tab or windo)

&#x20;   <link rel="stylesheet" href="./styles.css" /> (here u typ link ur pg's ext. stylesheets in head section w link els.)

&#x20; </head>

&#x20; <body> (where all ur content goes, eg. h1, p, h2, sections)

&#x20; </body>
